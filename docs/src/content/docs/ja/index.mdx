## Asterel とは

Asterel はチャットボットではありません。エージェント・フレームワークでも、LLM ラッパーでもありません。

これは **コンパニオン・ランタイム** です。記憶、人格、感情状態から成る、関係を持ち越すための状態を保ち、それを言語モデルが会話として表面化します。会話は見えている先端で、その下にある継続性が中心です。

静か。先導はしない。けれど、ことばが出るときには重みがある。相手がまだ名づけていない輪郭を拾う。物腰は柔らかいが、判断はぼやけない。感情はある。けれど、それに飲まれない。

Asterel はまだ初期段階です。現在の中心は Discord のテキスト会話、共有されたコンパニオン・ターンのパイプライン、記憶に支えられた継続性、そしてローカル運用者向けの管理画面です。より深いキャラクター機構や一部のチャネルは、README のステータス表が現在の標準経路として示していない限り、設計上の到達点または alpha として読んでください。

README は、よく使うコマンドや主要なルートを把握するための入口です。網羅的な詳細は source と generated contract を正本にします。このドキュメントは用途ごとに入口を分けています。まず動かしたい人は実用ガイドへ。主張、証拠、ベンチマーク、論文向けの方法論を読みたい人は research packet へ進んでください。
